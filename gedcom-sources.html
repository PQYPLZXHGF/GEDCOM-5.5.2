Abstract

This document lists known sources for GEDCOM files and, when possible, includes details about the program that uses the a given source.

This document is updated as new sources are discovered. Feedback and comments are welcome. You may create a pull-request to propose and collaborate on changes to the document.

GEDCOM Source List

When the VERS column has "5.5*", the file specifies HEAD.GEDC.VERS 5.5, but uses GEDCOM 5.5.1 features.

SOUR VERS Application
Ancestry.com Family Trees 5.5* Ancestry.com Family Trees
Ancestry.com Family Trees are GEDCOM files downloaded from Ancestry.com user-maintained family trees.
AncestQuest 5.5* Ancestral Quest
A genealogy application by Incline Software, LC; see https://www.ancquest.com
BROSKEEP 5.5* Brother's Keeper
A genealogy application by John Steed; see https://www.bkwin.org
Family Echo 5.5.1 Family Echo
An online applicaton by Familiality Ltd.; see https://www.familyecho.com
FAMILY HISTORIAN 5.5 Export Gedcom File (plug-in for Family Historian)
A plug-in for Family Historian that exports GEDCOM files subject to a variety of options; see https://www.family-historian.co.uk/pluginstore/plugin-entry?id=522
FAMILY_HISTORIAN 5.5 Family Historian
A genealogy application published by Calico Pie Limited; see https://www.family-historian.co.uk
FTM 5.5.1 Family Tree Maker for Windows
A genealogy application published by The Software MacKiev Company; see https://www.mackiev.com/
FTW 5.5 Family Tree Maker for Windows
A genealogy application published by multiple companies and superceded by the Software MacKieve version above
GEDitCOM 5.5 GEDitCOM
A genealogy application published by RSAC Software; see http://www.geditcom.com/
GENJ 5.5 GenealogyJ
Open source genealogy application; see https://sourceforge.net/projects/genj
Gramps 5.5.1 Gramps
Open source genealogy application; see https://github.com/gramps-project/gramps
GenoPro 5.5* GenoPro
Family tree and genogram creation tool; see http://www.genopro.com
(Creates invalid GEDCOM files that do not follow specifications for basic data, like marriages.)
HEREDIS PC2018
HEREDIS MAC2018		 5.5* Heredis
A genealogy application; see http://www.heredis.com
Heredis includes a version indicator in the SOUR value, so there are multiple other variations of the SOUR value. GEDCOM files with the following variations have been found online: HEREDIS PC2017, HEREDIS 12 PC, HEREDIS 11 PC, HEREDIS 8 PC, HEREDIS 7 PC, HEREDIS 6.1 PC
Legacy 5.5.1 Legacy
A genealogy application published by Millennia Corp.; see https://legacyfamilytree.com
MYHERITAGE 5.5 MyHeritage Family Tree Builder
See https://www.myheritage.com
PAF 5.5* Personal Ancestral File
See https://www.familysearch.org/paf
PAW2 5.5* Personal Ancestry Writer II
See http://lanopalera.org/Genealogy/AboutPAWriter.html
Reunion 5.5 Reunion for Mac
A genealogy application by Leister Productions; see https://www.leisterpro.com/
RootsMagic 5.5.1 RootsMagic
A genealogy application by RootsMagic, Inc.; see https://www.rootsmagic.com
rootstrust 5.5.1 rootstrust
A genealogy application by Atavus, Inc.; see https://www.rootstrust.com
The Next Generation of Genealogy Sitebuilding 5.5* The Next Generation of Genealogy Sitebuilding
A web-based genealogy application by Next Generation Software, LLC; see https://www.tngsitebuilding.com
TMG 5.5 The Master Genealogist
A genealogy application by Wholly Genes, Inc.; see https://www.whollygenes.com
webtrees 5.5.1 webtrees
An open source, web-based genealogy application; see https://www.webtrees.net/
